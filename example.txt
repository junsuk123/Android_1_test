1. 로컬 PC에  .git 폴더가 생성되었는지 확인 후  
2. 이름이 겹치지 않게 텍스트 파일을 작성
3. 원격 저장소(깃허브)에서 브랜치 추가(원하는 이름으로)
4.로컬에서도 브랜치 생성 (터미널에 위에서 만든 이름 그대로 브랜치 생성) ->git branch 브랜치명 
5. 생성한 브랜치를 현재 브랜치로 변경->git checkout 브랜치명
6. git add 파일이름.txt
7. git commit -m "작성하고 싶은 메세지"
8. git push origin 브랜치명
9. 깃허브 사이트 들어가서 내 푸시가 잘 들어갔는지 텍스트파일 및 내용 확인
